IBM-GENAI-PLANT-DISEASE-DETECTION

A machine learning-powered plant disease detection system that classifies plant diseases from uploaded leaf images. Built with TensorFlow, OpenCV, and Streamlit, it delivers fast and accurate predictions to promote sustainable farming and early disease management.
